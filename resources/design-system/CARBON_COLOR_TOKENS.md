## üéØ Semantic Color Tokens

### Interactive Colors

```css
/* Button, Link, Focus states */
--cds-interactive-01: #0f62fe;  /* Primary interactive */
--cds-interactive-02: #393939;  /* Secondary interactive */
--cds-interactive-03: #0f62fe;  /* Primary icons */
--cds-interactive-04: #0f62fe;  /* Selected elements */

/* Hover states */
--cds-hover-primary: #0353e9;
--cds-hover-secondary: #4c4c4c;
--cds-hover-tertiary: #0353e9;
--cds-hover-row: rgba(141, 141, 141, 0.12);
--cds-hover-selected-ui: #cacaca;

/* Active states */
--cds-active-primary: #002d9c;
--cds-active-secondary: #6f6f6f;
--cds-active-tertiary: #002d9c;

/* Disabled states */
--cds-disabled-01: #f4f4f4;
--cds-disabled-02: #c6c6c6;
--cds-disabled-03: #8d8d8d;
```

## üìê Usage Examples

### Button with Carbon Colors

```tsx
// Primary Button
<button className="bg-[#0f62fe] hover:bg-[#0353e9] active:bg-[#002d9c] text-white">
  Primary
</button>

// Danger Button
<button className="bg-[#da1e28] hover:bg-[#b81921] text-white">
  Delete
</button>

// Success Badge
<span className="bg-[#24a148] text-white px-2 py-1 text-xs">
  Success
</span>
```

### Text with Carbon Colors

```tsx
// Primary text
<h1 className="text-[#161616]">Heading</h1>

// Secondary text
<p className="text-[#525252]">Description</p>

// Placeholder
<span className="text-[#a8a8a8]">Placeholder</span>

// Error text
<span className="text-[#da1e28]">Error message</span>
```

### Backgrounds with Carbon Colors

```tsx
// Light theme page
<div className="bg-[#f4f4f4]">
  <div className="bg-white p-4">Card content</div>
</div>

// Dark theme sidebar
<aside className="bg-[#161616]">
  <nav className="bg-[#262626]">Nav content</nav>
</aside>
```
